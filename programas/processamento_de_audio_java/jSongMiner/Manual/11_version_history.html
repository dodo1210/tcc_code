Version History

Changes introduced in Version1.1.2:

  • Updated support libraries.
  • Minor documentation updates.

Changes introduced in Version1.1 (September 25, 2011):

  • Added a Music Brainz Java library
  • Updated the Echo Nest Java library
  • Added Music Brainz song fields: Song Title, Music Brainz Song ID, Artist Name, Music Brainz Artist ID, Album Title, Music Brainz Album ID, Song Release, Duration (seconds)
  • Added Music Brainz artist fields: Artist Name, Music Brainz Artist ID, Years Active, Lifespan, Artist Alias, Top Albums
  • Added Music Brainz album fields: Album Title, Music Brainz Album ID, Amazon Album ID, Release Type, Artist Name, Music Brainz Artist ID, Number of Tracks, Track List
  • Added Echo Nest song fields: 7digital-US Song ID, Echo Nest Release Name, Echo Nest Danceability (0 to 1), Echo Nest Energy (0 to 1), Echo Nest Analysis URL
  • Added Echo Nest artist fields: Rdio-US Artist ID, Facebook Artist ID, Years Active, Artist-Related Song
  • Updated unqualified and qualified Dublin Core mappings to reflect these new fields
  • Added the save_music_brainz_metadata tag to the configuration file format
  • Updated jMIR support libraries
  • A variety of small fixes
