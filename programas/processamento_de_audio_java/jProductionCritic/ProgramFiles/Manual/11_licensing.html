LIBRARIES USED BY JPRODUCTIONCRITIC

The following Java .jar files serve as jProductionCritic libraries, and are packaged with jProductionCritic:

  • ACE: A metalearning tool. jProductionCritic uses it to save ACE XML files. Like jProductionCritic, ACE is part of the jMIR framework.
  • UtilityClasses: A set of utility classes providing a variety of general-purpose functionality. jProductionCritic uses it to perform miscellaneous tasks, especially those relating to parsing audio files. Like jProductionCritic, these are part of the jMIR framework.
  • mp3plugin.jar: A JMF MP3 plugin distributed by Oracle used for parsing MP3 files.
